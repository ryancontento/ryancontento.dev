Introduction to Blazor

July 3, 2022

Blazor is Microsofts latest SPA (Single Page Application) framework. Blazor is open source and cross platform. If you love C#, .NET MVC, or just new and interesting frameworks, Blazor may be for you. What’s WebAssembly? WebAssembly (abbreviated Wasm) is a binary instruction format for a stack-based virtual machine. Wasm is designed as a portable compilation target for programming languages, enabling deployment on the web for client and server applications. source: webassembly.
